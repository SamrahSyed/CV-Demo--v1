Object Detection

Object recognition is a general term to describe a collection of related computer vision tasks that involve identifying objects in digital photographs. Object recognition consists of three in independent tasks: image classification, object localization, and object detection. Image classification involves predicting the class of one object in an image. Object localization refers to identifying the location of one or more objects in an image and drawing abounding box around their extent. Object detection combines these two tasks and localizes and classifies one or more objects in an image.
